## HR Analytics – Employee Attrition Project

This project analyzes employee attrition to identify key factors influencing employee turnover in organizations. It is designed to showcase skills in HR analytics, data visualization, and Python-based analysis for anyone starting a career in analytics.

## Tools & Technologies

Python
Pandas – Data manipulation and analysis
Matplotlib – Data visualization
Seaborn – Statistical data visualization

## Key Insights

Employees with low job satisfaction have higher attrition rates.
Work-life balance is a significant factor affecting employee retention.
Additional insights can be explored within the notebook to support HR decision-making.

## How to Use

Clone this repository:
git clone https://github.com/DipanjaliD/HR-Attrition-Project.git
Install required dependencies:
pip install pandas matplotlib seaborn
Open HR-Attrition-Project.ipynb in Jupyter Notebook.
Run the cells to explore the data analysis and visualizations.

## Purpose

This project demonstrates the ability to analyze HR data, identify trends, and provide actionable insights, helping organizations reduce employee turnover and improve workforce planning.
